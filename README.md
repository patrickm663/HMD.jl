# HMD.jl
Makes it easy to get data from [Human Mortality Database (HMD)](https://www.mortality.org/) into Julia!

## Installation
Clone the repo, enter and activate!

```bash
git clone git@github.com:patrickm663/HMD.jl
cd HMD.jl/
julia
```
```julia
julia> using Pkg
julia> Pkg.activate(".")
```
